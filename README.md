# 📋 Task Manager Web Application

A simple and responsive Task Manager Web Application built using **HTML**, **CSS**, and **JavaScript**. This project helps users manage their daily tasks efficiently by allowing them to add, complete, and delete tasks dynamically without reloading the page.

## 🚀 Live Demo



## 📂 GitHub Repository
https://github.com/shoubhyasinha21/Task-Manager-Web-project

## 📌 Features

* ✅ Add new tasks
* ✅ Display task list dynamically
* ✅ Mark tasks as completed
* ✅ Delete tasks
* ✅ Responsive user interface
* ✅ Real-time DOM manipulation
* ✅ No page reload required

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📁 Project Structure

```text
Task-Manager/
│
├── index.html
├── style.css
└── script.js
```

## ⚙️ How to Run the Project
1. Download or clone the project

2. Open the project folder.

3. Double-click `index.html` or open it using Live Server in VS Code.

4. Start managing your tasks.
